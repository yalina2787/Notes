# Create an GPG key

##Open terminal
1. Install Homebrew 
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
2. Check whether it is installed successfully or not. 
```
brew help
```
3. Install GPG
```
brew install gnupg
```
4. Check GPG version
```
gpg --version
```
